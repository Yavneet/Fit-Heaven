# Fit Heaven

Fit Heaven is a fitness web application designed to help users explore workout routines, gym schedules, memberships, and reward programs.

## Features
- Workout exercise page
- Membership plans
- Gym schedule
- Rewards system
- Location map for gym centers

## Tech Stack
HTML
CSS (Bootstrap)
JavaScript

## Project Purpose
This project was developed to practice frontend web development and build a structured multi-page fitness application.

## Future Improvements
- Add user login system
- Workout tracking
- Backend database integration
- Personalized fitness dashboard
